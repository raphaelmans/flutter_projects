### Setup
#### Clone the repo

```
$ git clone https://github.com/joshuadeguzman/flutter-examples
```
Select a project

```
$ cd <project_name>
```

Install dependencies

```
$ flutter packages get
```
Run the app

```
$ flutter run -v
```
